# Changelog

## [0.1.0] - 2026-01-17

### Added
- Initial Rust implementation of opencode
- Interactive TUI mode with ratatui
- Non-interactive prompt mode for scripting
- Support for Anthropic and OpenAI providers
- Configuration management with JSON/JSONC support
- Session management
- Built-in tools: read, write, edit, bash, glob, grep, todo
- TTY detection with helpful error messages
- Config initialization command
- Themes support (deltarune, undertale)
- Environment variable substitution in config
- Japanese README

### Features
- Fast, native Rust performance
- Single binary distribution
- Streaming LLM responses
- Tool integration for file operations
- Configuration layering (global + project)
- Multiple output formats (text, json, markdown)

### Documentation
- Comprehensive README in English and Japanese
- Example configurations in .opencode/
- Built-in help for all commands
